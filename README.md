# Company Management Suite

A management suite designed for small and medium businesses.

🔧 **Tech stack:** React, React Native, Vite, Node.js, Express, Socket.IO (WebSocket), Prisma, SQLite, JWT, AsyncStorage, Multer.

### System Structure

*   **Frontend:** Web Application
*   **Backend:** REST API & WebSockets
*   **Mobile:** Cross-platform App
*   **Database:** Relational Data Store
